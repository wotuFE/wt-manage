# wt-manage
沃土前端社区的管理系统

## 结构如下：(可能少了点什么,按需求再补充吧)
assets: 图片素材

components: 公用组件/封装组件 (ps: 每个组件独占一个文件夹)

views: 视图页面组件 (ps: 每一个页面组件独占一个文件夹,若有子页面,放在对应的children文件夹里)

router: 路由管理

store: vuex状态管理 (如有需要,可以分模块,模块放在modules文件夹里,文件名称最好与对应页面或模块同名)

styles: 样式文件

utils: 工具类/公共函数
